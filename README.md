# Curso de Lógica de Programação do EBAC

Aqui estão o passo a passo seguido nos estudos de lógica de programação em python do curso do EBAC

# Objetivos didáticos de cada aula:

## Aula 2 e 3 (1-2 Operadores_e_variaveis)
- conhecer a importância dos comentários em código Python;
- identificar os símbolos de operadores aritméticos e relacionais;
- compreender o raciocínio dos operadores lógicos.
- conhecer regras de nomenclatura de variáveis em Python;
- identificar os diferentes tipos de variáveis que podem ser usados em Python.

# Aula 4 (1_3_interação_com_usuário)
- identificar o uso do comando "print" para exibir informações na saída padrão;
- compreender como o comando "type" é usado para determinar o tipo de uma variável em Python;
- utilizar o comando "input" para interagir com usuário, coletar dados de entrada e armazená-los em variáveis.

# Aula 5 (1_4_estrutura_condicional)
- conhecer a estrutura das condicionais "if", "if/else" e "if/elif/else" em Python;
- compreender as ações das estruturas condicionais;
- descrever o uso dos operadores lógicos "and", "or" e "not" em estruturas condicionais.

# Aula 6 (1_5_listas_de_valores)
- compreender a estrutura de listas em Python;
- identificar as características de tuplas em Python;
- diferenciar a utilização de dicionários em Python.

# Aula 7 (1_6_estrutura_de_repeticao)
- compreender as estruturas de repetição "for" e "while" em Python e suas funções básicas em um programa;
- conhecer o uso da estrutura "for" com a função "range" em Python.
